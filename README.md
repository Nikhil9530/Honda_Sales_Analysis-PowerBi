# 📊 Honda Sales Dashboard Analysis (Power BI)

## 🔍 Project Overview
This project presents an interactive Power BI dashboard analyzing Honda bike sales performance across different regions, payment modes, and customer behavior. The goal is to derive meaningful insights from sales data to support business decision-making.

## 🛠 Tools & Technologies Used
- Power BI
- Excel (Data Source)
- Data Cleaning & Transformation
- Data Visualization

## 📈 Key Metrics Displayed
- Total Orders Count: 280
- Average Customer Rating: 4.30
- Net Sales: 27M
- Cost Price: 20M
- Gross Profit: 7M
- Finance Amount: 14M
- Insurance Amount: 2M

## 📊 Dashboard Features
- Interactive filters for **Payment Mode** (Cash, Credit Card, UPI, Finance)
- State-wise analysis (Maharashtra, Gujarat, Delhi, etc.)
- Monthly trends for:
  - Sales
  - Profit
  - Cost
- Product-level insights (Honda bike models)
- KPI cards for quick performance overview

## 🔎 Key Insights
- Finance and credit-based purchases contribute significantly to total sales
- Certain states show higher sales performance compared to others
- Monthly trends indicate fluctuations in sales and profit across the year
- High customer ratings indicate strong product satisfaction
